Утиліта для детектування приладів з однаковим номером в РКОМі.
Запускається на хості з РКОМом.
РКОМ та Монго повинні бути запущені.

У випадку, якщо даною програмою виявлено прилади з однаковими номерами, необхідно дані номера видалити з бази РКОМу за допомогою утиліти "delete-ppk-rcom-x86".
Якщо в базі присутні записи з однаковими номерами, мобільний додаток для таких приладів може працювати не коректно.
Версія РКОМ v.2.7.9 і вище забороняє створення в базі записів з однаковими номерами.
